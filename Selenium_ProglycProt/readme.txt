Overview

This script automates the process of checking protein sequences in the ProglycProt database using Selenium for web automation. It retrieves UniProt sequences, verifies glycosylation sites, and updates a CSV file with the results.

Features

Automates navigation and data retrieval from the ProglycProt website.

Searches for UniProt IDs and retrieves protein sequences.

Checks glycosylation site mutations.

Updates a CSV file (check_output_file.csv) with the validation results.

Handles cases where sequences are not found or are empty.
